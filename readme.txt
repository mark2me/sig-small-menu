== Description ==

利用佈景的自訂選單製作出側邊和手機版底部的固定位置選單。

== Installation ==

1. 上傳到外掛資料夾下 `/wp-content/plugins/sig-small-menu` 。
1. 在 WordPress 後台的 `外掛` 頁面啟用此掛。
